#  Welcome to <a>CV Zaman Now</a><br/> Design System


‚ö° Build your own experiences on the CV Zaman Now platform!<br/>
This framework provides low level utilities and UI components for building custom experiences.


<br/>

## Style

The CVZamanNow Design System stylesheet also provides a set of global CSS classes that can be used for typesetting and customizing other style properties. 
All CSS classes provided by this framework implement standard values.

<br/>

## Components

UI Components are currently offered as React (Web) components. Direct control over styling is limited by design, but theming is supported.

<br/>


## Installation

Make sure you have Node.js and npm installed on your computer before starting. If not, you can download it from [Node.js website](https://nodejs.org/).

To install the CV Zaman Now package, run the following command in the terminal :

```bash
npm i @cvzamannow/uikit
```
NPM package link: [NPM Package cvzamannow‚ú®](https://www.npmjs.com/package/@cvzamannow/uikit)

<br/>


## Usage Example

After the package is installed, you can create your own file to create a CV using the components provided by CV Zaman Now.

Create a file with the extension `.tsx` or `.ts` (eg FormPage.tsx), for example like this:

```tsx
import { Header, Button, Form } from '@cvzamannow/uikit';

const FormPage = () => {
  return (
    <div>
      <Header
        NavLogo={[

        ]}
        ProfileWrapper={[
          <><Button ButtonType="nolo" label="Daftar" style={{height: '48px', width: '118px'}}/><Button ButtonType="text" label="Masuk" style={{marginLeft: '20px'}}/></>

        ]}
```
